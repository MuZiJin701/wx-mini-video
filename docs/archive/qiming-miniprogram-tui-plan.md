# 启明星小程序视频下载 TUI 重构计划

## Summary

把当前项目收敛成一个专用工具：只服务“启明星软件”小程序，启动本地 HTTPS 代理嗅探 PC 微信小程序请求，在 TUI 中实时显示识别到的视频资源，支持直链视频下载，识别到 `m3u8` 时调用本机 `ffmpeg` 合并输出为 `.mp4`。

默认采用“代码级精简”：删除视频号、公众号、文件助手、Web API、旧注入脚本、旧文档和无关 CLI，只保留小程序嗅探、代理证书、系统代理、下载、配置、TUI 所需代码。

## Key Changes

- `go run .` 直接进入 TUI，不再保留旧的 `channels`、`mp`、`sph`、`server`、`download` 等命令。
- 固定目标小程序：`启明星软件`，AppID `wx9ed9ca51b87086d9`，原始 ID `gh_f8465291ff78`。
- 保留并整理小程序候选资源识别逻辑，支持从响应 URL 和 JSON 响应中提取 `mp4/m3u8/webm/mov`。
- 普通视频直链使用 Go 原生 HTTP 下载，并复用嗅探到的请求头。
- `m3u8` 资源调用本机 `ffmpeg` 合并输出为 `.mp4`。
- TUI 使用 Bubble Tea 生态展示资源列表、详情、下载队列和日志。
- 工程结构精简为小程序嗅探、代理、下载、配置、TUI 几个小模块。

## TUI Behavior

- 启动后加载配置、初始化证书、启动代理，并按配置设置系统代理。
- 界面提示用户在 PC 微信中打开启明星软件小程序并播放视频。
- 资源列表实时刷新，展示类型、大小、来源字段、创建时间和 URL。
- 快捷键：
  - `r` 刷新候选资源
  - `d` 下载当前选中资源
  - `c` 清空候选资源
  - `o` 打开下载目录
  - `l` 切换日志面板
  - `q` 退出并恢复系统代理

## Test Plan

- 单元测试：
  - 资源识别：`mp4`、`m3u8`、`webm`、无关 URL、JSON 嵌套字段。
  - 请求头清洗：过滤 `Host`、`Content-Length`、`Accept-Encoding` 等。
  - 文件名生成：候选 ID、URL basename、m3u8 强制 `.mp4`。
  - `ffmpeg` 命令构建：headers 拼接、路径处理、输出扩展名。
- 集成验证：
  - `go test ./...`
  - `go build -o qiming-video.exe .`
  - 手动启动 TUI，确认代理启动、退出恢复代理。
  - 在 PC 微信启明星小程序中播放视频，确认 TUI 出现候选资源。
  - 下载直链视频和 `m3u8` 视频。

## Assumptions

- 代码级精简会实际删除旧功能代码，而不是只隐藏入口。
- `m3u8` 下载依赖本机 `ffmpeg`，程序只检测和调用，不内置 ffmpeg。
- 分享链接本身仍不直接解析视频地址；可靠路径是启动代理后在 PC 微信小程序内播放视频进行嗅探。
